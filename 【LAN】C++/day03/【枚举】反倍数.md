# 一、题目描述

> 给定三个整数 $a, b, c$，如果一个整数既不是 $a$ 的整数倍，也不是 $b$ 的整数倍，还不是 $c$ 的整数倍，则这个数称为反倍数。
> 请问在 $1$ 至 $n$ 中有多少个反倍数？
## 1.1 输入描述
> 输入的第一行包含一个整数 $n$。  
> 第二行包含三个整数 $a, b, c$，相邻两个数之间用一个空格分隔。  
> 其中，$1 \leq n \leq 1000000$，$1 \leq a \leq n$，$1 \leq b \leq n$，$1 \leq c \leq n$。
## 1.2 输出描述
> 输出一行包含一个整数，表示答案。
## 1.3 输入输出样例
- 输入

```
30  
2 3 6
```
- 输出

```
10
```
**样例说明**：以下这些数满足要求：1, 5, 7, 11, 13, 17, 19, 23, 25, 29。
**原题链接**：[反倍数](https://www.lanqiao.cn/problems/152/learning/?page=1&first_category_id=1&problem_id=152)
# 二、解题思路
## 2.1 读取数据：
> 程序先==接收任务参数==，首先读入一个整数 `n`，比如输入 `30`，这表示我们要在 `1` 到 `30` 这个范围内找反倍数。接着读入三个整数 `a`、`b`、`c`，比如输入 `2 3 6`，这三个数是判断倍数的==标准==。  
## 2.2 逐个检查数字：
从 `1` 开始，一个一个看遍 `1` 到 `n` 的所有数字，比如检查数字 `5`：
- 看 `5` 能不能被 `a`（2）整除？不能；
- 看 `5` 能不能被 `b`（3）整除？不能；
- 看 `5` 能不能被 `c`（6）整除？还是不能

这时候，`5` 既不是 `a` 的倍数，也不是 `b` 和 `c` 的倍数，符合反倍数条件，于是启动一个==计数器== `sum`，让 `sum` 的值加 `1`
> 对每个数字都重复这个检查过程：只要数字不是 `a`、`b`、`c` 中任意一个数的倍数，就给 `sum` 加 `1`。
## 2.3 输出结果：
> 当检查完 `1` 到 `n` 的所有数字后，`sum` 里存的就是 `1` 到 `n` 中反倍数的总数量。最后把 `sum` 的值输出，这就是题目要求的答案。
# 三、代码详解

```cpp
#include <bits/stdc++.h>
using namespace std;

bool iftimes(int a, int b, int c, int x){
  if(x % a == 0 || x % b == 0 || x % c == 0){
    return true;
  }
  return false;
}

int main()
{
  int n;
  cin >> n;

  int a, b, c;
  cin >> a >> b >> c;

  int sum = 0;
  for(int i = 1; i <= n; i++){
    if(iftimes(a, b, c, i) == false){
      sum += 1;
    }
  }

  cout << sum << '\n';

  return 0;
}
```
## 3.1 iftimes函数：
> 检查是否是 a/b/c 的倍数。
## 3.2 主循环：
> 逐个检查每个数。
## 3.3 条件判断：
> 用取模运算判断倍数关系。
