# 一、题目描述
> 小明对数位中含有 2、0、1、9 的数字很感兴趣（不包括前导 0），在 1 到 40 中这样的数包括 1、2、9、10 至 32、39 和 40，共 28 个，他们的和是 574。
> 请问，在 1 到 n 中，所有这样的数的和是多少？
## 1.1 输入描述
> 输入一行包含两个整数 `n`（`1 ≤ n ≤ 10⁴`）。
## 1.2 输出描述
> 输出一行，包含一个整数，表示满足条件的数的和。
## 1.3 输入输出样例
- 输入

```
40
```
- 输出

```
574
```
原题链接：[特别数的和](https://www.lanqiao.cn/problems/191/learning/?page=1&first_category_id=1&problem_id=191)

# 二、解题思路
> 咱们要解决的问题是：在 1 到 n 这些数字里，找到所有 “数位中包含 2、0、1、9 任意一个” 的数，然后把这些数加起来，求总和。
## 第一步：判断单个数字是否符合条件
比如现在有个数 19，怎么知道它符不符合？
- **拆数位**：把数字从右到左一位一位拆开看。用==取余数==的方法，每次拿最后一位。比如 19 % 10 得到 9，发现 9 是目标数字，立刻知道这个数符合条件
- **循环检查**：如果最后一位不是目标数字，就把最后一位==扔掉==（用除法，比如 19 / 10 变成 1），继续检查剩下的数位。直到所有数位查完。如果全程没找到目标数字，这个数就不符合条件

举个例子：
- 数字 3：3 % 10 得到 3，不是 2、0、1、9，扔掉后没数位了，所以 3 不符合
- 数字 10：10 % 10 得到 0，是目标数字，立刻判定 10 符合条件
## 第二步：遍历 1 到 n 所有数字，累加符合条件的数
从 1 开始，一个一个数检查：
- 对每个数字，用第一步的方法判断是否符合条件
- 如果符合，就把这个数字加到总和里
- 全部检查完后，总和就是最终结果

比如输入 n=40：
- 从 1 查到 40，像 1 符合（个位是 1），2 符合（个位是 2），9 符合……一直查到 40。每查到一个符合条件的数，就加到总和里，最后得到 574。

> 总结起来：先写个==判断数字是否符合条件==的工具函数，再从头到尾查一遍 1 到 n 的数，符合条件的就累加，最后输出结果。
# 三、代码详解

```cpp
#include <iostream>
using namespace std;
bool is2019(int n){
  while(n){
    int x = n % 10;
    if(x == 2 || x == 0 || x == 1 || x == 9){
      return true;
    }
    n = n / 10;
  }

  return false; 
}

int main()
{
  int n;
  cin >> n;

  int sum = 0;
  for (int i = 1; i <= n; i++){
    if(is2019(i) == true){
      sum += i;
    }
  }

  cout << sum << '\n';

  return 0;
}
```
## 3.1 is2019 函数
- 通过`while`循环逐位拆解数字，`n % 10`获取末位数字
- 检查末位是否是 2、0、1、9，若是则返回`true`
- `n = n / 10`去掉已检查的末位，继续检查高位，直到所有位检查完毕
## 3.2 main 函数
- 输入 n，初始化总和`sum`为 0
- 遍历 1 到 n 的每个数字，用`is2019`判断是否符合条件，符合则累加
- 最后输出总和
